Python Hangman Game

Description
This is a terminal-based Hangman game built using Python.
The player guesses letters to uncover a hidden word within a limited number of attempts.

This project was created to practice Python fundamentals through game logic and user interaction.


Features
- Random word selection
- Letter guessing system
- Limited number of attempts
- Tracks guessed letters
- Displays word progress after each guess



Concepts Used
- Variables & data types
- Lists
- Strings
- Loops
- Conditional statements
- Functions
- Random module
